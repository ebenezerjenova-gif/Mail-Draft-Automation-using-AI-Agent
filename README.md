# 🤖 AI Agent-Based Mail Automation using n8n & Gemini AI

This project showcases an advanced mail automation workflow built using **n8n**, powered by an **AI Agent integrated with Gemini AI**. The workflow is triggered through a chat interface, where the AI Agent intelligently handles the entire process—from fetching data to generating and drafting emails.

## 🚀 Features
- 💬 Chat-triggered workflow execution  
- 📊 Fetches recipient data from Google Sheets  
- 🧠 Uses Gemini AI with memory for context-aware responses  
- ✉️ Generates personalized email content dynamically  
- 📩 Automatically creates Gmail drafts ready to send  

## 💡 Key Highlights
- Demonstrates the use of **AI Agents in automation**
- Combines **LLM + Memory + Tools** for intelligent workflows  
- Enables **end-to-end automation** of email communication  

## 📌 Use Case
Ideal for automating personalized email communication, reducing manual effort, and improving efficiency in real-world scenarios.
